## Variant: RoomScope Field Workflow

### Design stance
A practical iPad-first field workflow that keeps rooms, system cards, estimator flags, and PDF preview visible at the same time.

### Key choices
- Layout: three-pane iPad/tablet layout with project context left, active room center, flags/PDF preview right.
- Typography: system Apple-style fonts for a native iOS feel.
- Color: teal/blue for professional field-tool feel; amber for estimator flags.
- Interaction: buttons show visible toast feedback for estimator flag, photo, PDF inclusion, and export actions.

### Trade-offs
- Strong at: communicating the core workflow quickly; showing why this replaces Apple Notes.
- Weak at: small iPhone ergonomics; this variant is intentionally iPad-first.

### Best for
- Sales/designer walkthroughs where the user has an iPad and needs fast structured capture plus report confidence.
